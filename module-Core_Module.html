Module: Core/Module

Класс является родителем для всех модуле кроме главного модуля (Root Module).
Наследуется от класса Observable для реализации паттерна Наблюдатель (Observer).
Объект модуля реализует паттерн Одиночка (Singleton).
Кадый модуль создается только 1 раз, последующий вызов new Module() - вернет текущий экзепляр данного класса.
